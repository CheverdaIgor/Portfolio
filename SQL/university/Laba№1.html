Лабораторная работа №1

Знакомство с клиентской частью СУБД MS SQL. Создание базы данных.

Цель работы: научиться устанавливать соединение с MS SQL-сервером при помощи Enterprise Manager, уметь создавать базу данных, таблицы и строить диаграмму базы в Enterprise Manager.

Порядок выполнения работы:

  1. В окне Enterprise Manager выбрать пункт меню Действие (Action) -> New SQL Server Registration
  2. В мастере регистрации выбрать из доступных серверов сервер TEYA, на следующей страничке указать подключение с использованием The Windows account information I use to log on to my computer (использование имени и пароля, которые используются для работы с ОС) и затем вариант Add the SQL Server to an existing SQL Server group.
  3. Раскрыть дерево обьектов SQL Server и в нем выбрать закладку Databases (базы данных).
  4. Вызвать контекстное меню и выполнить команду New Database (новая БД...) после чего появится окно построителя баз данных, в котором следует указать такие параметры Вашей базы данных:
      в поле Name укажите свое имя пользователя - это будет имя базы данных;
      в области File growth (увеличение размера файла) выберите альтернативу In megabytes и установите значение параметра в 1.
      в области Maximum file size ограничьте размер базы данных Restrict filegrowth(MB)=5 .

    Все базы данных хранятся на сервере и все, что превысит 5 MB, будет уничтожаться!!!

  5. Для того, чтобы создать новую табличку, нужно в своей базе данных выбрать закладку Tables (таблицы), вызвать контекстное меню и выбрать пункт New Table (новая таблица).
  6. При создании таблиц может возникнуть необходимость использования автоинкрементного поля (поле, которое заполняется автоматически с указанным приростом, начиная с установленного значения). Для задания такого поля в разделе свойств этого поля устанавливают свойство Identity = Yes .
  7. Для создания ключевого поля в таблице следует спозиционировать курсор в данном поле и в контекстном меню воспользоваться командой Set primary key , либо изображением ключа на панели инструментов окна программы.
  8. После создания всех таблиц Вашей базы данных (Но перед заполнением таблиц!!!) для обеспечения целостности данных все таблицы нужно связать в диаграмму. Для этого выбирается закладка Diagrams, вызывается контекстное меню и выполняется команда New database diagram. При построении диаграммы Вы используете только собственноручно созданные таблицы. В окне диаграммы нужно связать таблицы по ключевым полям.
  9. Заполнить созданные таблицы данными (не менее 10-ти записей в каждой таблице). Для заполнения таблицы данными нужно выбрать таблицу, вызвать на ней контекстное меню и выполнить команду Open table -> Return all rows .
